# 竞品内容规律分析框架

## 内容类型分类

| 类型 | 英文 | 描述 | 典型互动特征 |
|---|---|---|---|
| 产品硬广 | Product Showcase | 直接展示产品，无故事 | ER 通常最低 |
| BTS 幕后 | Behind-the-Scenes | 制作过程、团队日常 | ER 较高，建立信任 |
| 社区关怀 | Community | 公告、感谢、节日问候 | ER 最高（粉丝感到被重视）|
| UGC 转发 | User Generated | 转发顾客打卡内容 | ER 高，社交证明 |
| 季节/节日 | Seasonal/Holiday | 节日限定、季节新品 | 搜索流量大 |
| 互动投票 | Engagement Poll | 提问、投票、挑战 | 评论数高 |
| 教育科普 | Educational | 成分解析、喝茶好处 | 转发率高 |
| 团队故事 | Team Story | 员工介绍、幕后团队 | 人格化，建立粘性 |
| 服务推广 | Service | 团餐、外卖、预约 | B2B 转化 |
| KOL 合作 | Influencer | 博主探店、合作内容 | 曝光大，信任度高 |

## ER（互动率）分析

```
ER = (Likes + Comments) / Followers × 100%
```

### ER 基准参考

| ER 区间 | 评级 | 说明 |
|---|---|---|
| > 8% | 极佳 | 通常是小账号（<5K）有高度粘性粉丝 |
| 4% - 8% | 好 | 健康区间，说明内容有共鸣 |
| 1% - 4% | 一般 | 多数账号在此区间 |
| < 1% | 低 | 需要优化内容策略 |
| < 0.3% | 警告 | 账号可能已停止活跃运营，或在买粉 |

## 常见内容公式（Caption 结构）

### 竞品最常用的公式 A：Hook + 故事 + CTA
```
[一句抓眼球的话，不超过10个字]
[2-3句背景/故事/成分描述]
[一句软 CTA]
#[tag1] #[tag2] #[tag3]
```

### 竞品最常用的公式 B：价值主张 + 细节 + CTA
```
[核心卖点一句话]
[1-2句详细描述，包含食材/工艺/感受]
[互动引导或到店邀请]
#[tag1] #[tag2]
```

### 竞品最常用的公式 C：BTS 叙事
```
[描述一个动作或场景开头]
[透露一个幕后细节]
[用一句总结收尾]
#[tag1] #[tag2]
```

## Hashtag 策略层级

| 层级 | 类型 | 示例 | 数量 |
|---|---|---|---|
| 品牌层 | 自有品牌标签 | #BubbleU #品牌名 | 1-2 |
| 地理层 | 本地精准 | #DTLAeats #SGVFood | 1-3 |
| 品类层 | 垂直社区 | #BobaLovers #CraftBoba | 2-4 |
| 流量层 | 泛流量 | #Foodie #InstaFood | 1-2 |
| 竞品层 | 蹭流量 | #[竞品城市]eats | 0-1 |

## 差距分析模板

```markdown
## 本品差距分析

### 高优先级（差距大 + 门槛低）
1. **[内容类型]**：竞品X/Y都在做，BubbleU 没有 → 立即做
2. **[内容类型]**：竞品有但频率低，可以抢占 → 快速做

### 中优先级（差距中 + 需要投入）
3. **[内容类型]**：需要视频制作投入，但竞品验证有效 → 规划做
4. **[Hashtag]**：竞品共用某标签，BubbleU 未使用 → 加入清单

### 低优先级（差距小 or 门槛高）
5. **[内容类型]**：竞品在做但效果一般 → 观望
6. **[内容类型]**：需要大预算或外部资源 → 长期规划
```
