# 合规审核规则

## Pre-Publish Checklist（每帖必查）

### Copy 合规

| 检查项 | 规则 | 违规处理 |
|---|---|---|
| Caption 长度（IG） | ≤ 2,200 chars | 截断或删减 |
| Caption 长度（TikTok） | ≤ 2,200 chars | 同上 |
| Caption 长度（FB） | ≤ 63,206 chars | 几乎不会超 |
| Hashtag 数量（IG） | 5-15 个 | 删减到范围内 |
| Hashtag 数量（TikTok） | 3-5 个 | 精选高频标签 |
| Hashtag 数量（FB） | 1-3 个 | 精选 |
| CTA 存在 | 必须有软 CTA | 添加 |
| 无违禁词 | 见下方违禁词表 | 替换或删除 |

### Visual 合规

| 检查项 | 规则 | 违规处理 |
|---|---|---|
| 图片文字占比 | ≤ 20%（FB/IG 广告规则）| 减少文字覆盖面积 |
| Logo 存在 | 推荐有 | 建议添加 |
| Logo 位置 | 右下角为佳 | 标注供用户确认 |
| 无版权图片 | 不得含第三方品牌素材 | 替换 |

### 促销/ giveaway 合规（⚠️ 美国 FTC 规则）

促销内容必须包含以下之一：
- **有限数量**："前50名顾客" → 需附法律文本
- **有时间限制**："仅今天" → 需附截止日期
- **有购买条件**："买一送一" → 需附条款

**法律文本模板（有限数量促销）：**
```
*First {N} customers on {date} at {location} receive {reward}. While supplies last. 
No purchase necessary. Not valid with other offers. Not transferable.
{BrandName} reserves the right to modify or cancel this promotion.
```

**法律文本模板（买一送一等促销）：**
```
*Buy one, get one 50% off on {date} at {location}. While supplies last. 
Valid on regular-priced items only. Not valid with other offers. 
Not redeemable for cash.
```

**简化版（用于 caption 末尾）：**
```
While supplies last. Not valid with other offers.
```

### F&B（餐饮）特定规则

| 规则 | 说明 |
|---|---|
| 不得声称"治愈"疾病 | 不能写"our tea cures..." |
| 营养声明需有依据 | 不能无凭据写"无糖/低卡" |
| 价格信息必须准确 | 不得写过期价格 |
| 赠品/竞赛必须注明条款 | 见上方促销合规 |

## 违禁词表

### Instagram / Facebook

```
# 绝对禁止
free money, make money fast, click here now, guaranteed income
no obligation, risk free, 100% free, act now
DM for price, negotiate, limited time offer (without date)

# 高风险（需确认上下文）
FREE (大写单独出现，可能是促销违规)
WIN (竞赛必须有法律文本)
CONTEST (同上)
GIVEAWAY (同上)
```

## 评级处理规则

### 🔴 BLOCK — 必须修复

含以下内容的帖子不得发布：
- 未附法律文本的促销
- 健康/医疗虚假声明
- 版权侵权内容
- 仇恨/歧视性内容

**处理方式：** 修改文案或移除相关内容，给出具体修改方案。

### 🟡 WARN — 建议修复

含以下内容的帖子可发布，但建议优化：
- Hashtag 超出建议数量
- Caption 接近字数上限
- 图片文字面积偏大
- 轻微语法/拼写问题

**处理方式：** 给出优化建议，用户确认后更新。

### ✅ PASS — 直接发布

通过所有检查项的内容。
